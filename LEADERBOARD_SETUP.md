# Shared Leaderboard Setup

This project is ready for a shared leaderboard using Supabase + Vercel.

## 1. Create a Supabase project

Create a new project at:

- `https://supabase.com`

## 2. Create the `scores` table

Run this SQL in the Supabase SQL editor:

```sql
create table if not exists public.scores (
  id bigint generated by default as identity primary key,
  name text not null,
  score integer not null,
  rounds integer not null default 8,
  created_at timestamptz not null default now()
);
```

## 3. Add environment variables in Vercel

In your Vercel project, add:

- `SUPABASE_URL`
- `SUPABASE_SERVICE_ROLE_KEY`

Use the values from your Supabase project settings.

## 4. Deploy

After deploying, the game will use:

- `GET /api/leaderboard`
- `POST /api/leaderboard`

If the leaderboard is empty, it will automatically seed starter scores.
