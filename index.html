How To Play

Rules

Every board starts with three highlighted wheel letters. Your words must include all three letters, and letters can repeat.

Arcade mode is the timed score chase: solve one 4-letter, 5-letter, 6-letter, and 7-letter word before the timer expires, then roll into the next board.

Puzzle mode is the hidden-answer board: solve alphabetized mystery words in the 5-letter, 6-letter, 7-letter, and 8-letter groups before the 2 minute round timer runs out.

Theme mode builds a themed hidden board first, then rotates the wheel to real 3-letter slices from the remaining answers so you can chase down the themed board before the 2 minute timer expires.

In multiplayer, both players share the same board. Ready up, race through eight rounds, and the higher score wins.

Triple Stack

Readme

Triple Stack gives you one pool of letters and three word slots to fill from smallest to largest.

Use each letter only once. Every submitted word locks into the next slot, and the full set is checked after the third word is entered.

You get three lives. A life is only lost when a full three-word set is submitted and it still does not solve the puzzle.

You can delete letters while building a word, reset the whole puzzle up to three times, or give up to reveal the answer.

No score pressure here. It is a pure solve-the-stack puzzle.

Single Player

LexiPath

Build a chain from the start word to the end word by following the colored arrows.

Type freely into the intermediate slots, press Enter to lock a word into place, and only press Submit Chain when you want the whole path checked.

The path can follow the move rules and still miss the authored answer. You only win when the full chain matches the intended stored sequence.

LexiPath

Readme

LexiPath is a structured word ladder. Each arrow tells you how the next word must change.

Blue means rearrange only. Yellow means add one letter without changing the original order. Green means add one letter and rearrange. Red means replace one letter and rearrange.

You can try any word that fits the rule. The puzzle only checks for the exact intended path when you press Submit Chain.

Use Enter to lock each intermediate word into its slot, then submit when the whole path feels ready.
